### Fixed
- **Client emails now actually deliver.** Revision/accepted/declined confirmations, quote reminders and contract renewal emails resolved the recipient from the quote/contract row's own email — which is usually blank (the address lives on the linked client record), so they were silently skipped. A new `recipient_for()` fallback resolves the client's email at every send site.

### Added
- **"Quote — revised (re-send)"** template — once a quote has been through a client revision, the Send modal seeds this distinct copy instead of the original first-send message.
- Client email sends now appear in the quote **Activity** timeline (alongside the Delivery log).

### Changed
- **Send quote** message body is a formatted rich-text editor (sans-serif, no Code tab).
- **"Reopen as draft to revise"** opens the quote editor directly.
- **Mobile:** quote and contract client pages stack line items into cards (labeled Unit / Qty / Total) so the table never overflows; quote meta block left-aligned.
